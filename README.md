## Getting Started

### Prerequisites

- [Terraform](https://www.terraform.io/downloads.html) >= 1.0.0
- AWS Credentials configured (e.g., `~/.aws/credentials` or environment variables)

### Usage

1.  **Initialize Terraform**:
    ```bash
    terraform init
    ```

2.  **Plan the infrastructure**:
    ```bash
    terraform plan
    ```

3.  **Apply the changes**:
    ```bash
    terraform apply
    ```
